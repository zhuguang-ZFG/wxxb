### Markdown 格式

所有文档使用 Markdown 格式：

- 易于编辑
- 版本控制友好
- 可转换为其他格式
- 支持代码高亮

### 格式规范

1. **标题层级**
   - H1: 文档标题
   - H2: 主要章节
   - H3: 子章节
   - H4: 小节

2. **代码块**
   ```bash

   pip install linktunnel
   ```

3. **表格**
   | 列1 | 列2 |
   |-----|-----|
   | 值1 | 值2 |

4. **列表**
   - 无序列表
   1. 有序列表

5. **强调**
   - **粗体**: 重要内容
   - *斜体*: 术语
   - `代码`: 命令和代码
