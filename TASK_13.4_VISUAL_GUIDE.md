## URL 生成示例

### Yacd 在线面板

**输入：**
- API: `http://127.0.0.1:9090`
- Secret: `my_secret_key`

**输出：**
```
https://yacd.metacubex.one/?hostname=127.0.0.1&port=9090&secret=my_secret_key
```

**特点：**
- 使用在线托管的 Yacd 面板
- 通过 URL 参数传递连接信息
- 适合快速访问，无需本地配置

### 本地 UI

**输入：**
- API: `http://127.0.0.1:9090`

**输出：**
```
http://127.0.0.1:9090/ui
```

**特点：**
- 使用 Mihomo 内置的 UI
- 需要在配置文件中设置 `external-ui` 路径
- 适合离线使用或自定义 UI

## 使用场景

### 场景 1：快速访问 Yacd

1. 用户配置好 API 和 Secret
2. 点击"打开控制台"
3. 选择"Yacd (在线)"
4. 浏览器自动打开 Yacd，无需手动输入参数

### 场景 2：使用本地 UI

1. 用户在 Mihomo 配置中设置了 `external-ui`
2. 点击"打开控制台"
3. 选择"本地 UI"
4. 浏览器打开本地托管的 UI

### 场景 3：从 profile 读取配置

1. 用户勾选"从默认 profile 的 config.yaml 读取 API / Secret"
2. 点击"打开控制台"
3. 自动从配置文件读取 API 和 Secret
4. 选择控制台类型并打开
