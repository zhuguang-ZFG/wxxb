## 已知限制

### 功能限制
- tkinter 版本功能有限（仅备选）
- 某些功能需要 PyQt6
- 网络功能需要网络连接

### 平台限制
- 主要支持 Windows/Linux/macOS
- 某些功能可能因平台而异
- 串口功能需要硬件支持

### 性能限制
- 大量节点验证可能较慢
- 大文件传输可能占用内存
- 实时监控可能影响性能

## 使用指南

### 快速开始
```bash
# 安装
pip install -e .

# 运行
python -m linktunnel.unified_gui

# 测试
pytest tests/
```

### 主要功能
1. **Grbl 控制**: 连接设备 → 选择命令 → 发送
2. **代理管理**: 配置 API → 选择节点 → 切换模式
3. **节点管理**: 添加节点 → 验证 → 自动更新
