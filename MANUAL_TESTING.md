## 🚀 快速测试

### 前提条件

1. **安装 Python 3.8+**
   ```bash
   python --version
   ```

2. **安装依赖**
   ```bash
   # 进入项目目录
   cd /path/to/linktunnel
   
   # 安装依赖
   pip install -e '.[gui-full]'
   ```

### 启动应用

#### 方式 1: 使用测试脚本（推荐）

```bash
python test_startup.py
```

这个脚本会：
- 检查 Python 版本
- 验证 PyQt6 安装
- 测试所有核心模块
- 启动应用程序

#### 方式 2: 直接启动

```bash
python -m linktunnel.unified_gui
```

#### 方式 3: 使用命令行入口

```bash
linktunnel-unified
```

## 🐛 常见问题

### 问题 1: 应用无法启动

**症状**: 双击或运行命令后无反应

**解决方法**:
1. 检查 Python 版本: `python --version`
2. 检查 PyQt6 安装: `pip list | grep PyQt6`
3. 查看错误信息: `python -m linktunnel.unified_gui`
4. 查看日志文件

### 问题 2: 导入错误

**症状**: `ModuleNotFoundError: No module named 'linktunnel'`

**解决方法**:
```bash
# 确保在项目根目录
cd /path/to/linktunnel

# 安装项目
pip install -e .

# 或使用完整路径
PYTHONPATH=src python -m linktunnel.unified_gui
```

### 问题 3: PyQt6 未安装

**症状**: `ImportError: No module named 'PyQt6'`

**解决方法**:
```bash
pip install PyQt6
# 或
pip install 'linktunnel[gui]'
```

### 问题 4: 窗口显示异常

**症状**: 窗口布局混乱或控件显示不正常

**解决方法**:
1. 删除配置文件
2. 重启应用
3. 尝试切换主题

### 问题 5: 性能问题

**症状**: 应用运行缓慢或卡顿

**解决方法**:
1. 清空日志查看器
2. 关闭不使用的模块
3. 检查系统资源
4. 重启应用
