## 📋 功能概览

Grbl 模块现在包含了一个完整的命令参考功能，可以在应用程序中直接查看和使用 Grbl 命令。

### 新增功能

1. ✅ **命令参考标签页**
   - 显示所有 Grbl 命令
   - 分类显示（系统命令、G代码、M代码、参数设置）
   - 每个命令都有详细说明

2. ✅ **命令搜索**
   - 实时搜索命令
   - 支持按命令名或说明搜索
   - 快速定位需要的命令

3. ✅ **命令详情**
   - 显示选中命令的详细说明
   - 一键发送命令

4. ✅ **快速发送**
   - 从命令列表直接发送命令
   - 无需手动输入

## 🚀 使用方法

### 方法 1: 浏览命令列表

1. 打开 Grbl 模块
2. 点击"命令参考"标签页
3. 查看所有可用命令
4. 点击命令查看详细说明

### 方法 2: 搜索命令

1. 在搜索框输入关键词（如 "回零" 或 "$H"）
2. 列表自动过滤显示匹配的命令
3. 点击命令查看详细说明

### 方法 3: 快速发送命令

1. 在命令列表中选中要发送的命令
2. 点击"发送选中命令"按钮
3. 命令自动发送到设备

### 方法 4: 手动输入命令

1. 在"手动控制"标签页的命令输入框输入命令
2. 按 Enter 或点击"发送"按钮
3. 命令发送到设备

```python
        "G1": "直线切割 - 以指定速度切割到指定位置",
        ...
    },
    ...
}
```

### 搜索功能

```python
def search_commands(keyword: str) -> list[tuple[str, str, str]]:
    """搜索命令"""
    results = []
    keyword = keyword.lower()
    for category, commands in GRBL_COMMANDS.items():
        for cmd, desc in commands.items():
            if keyword in cmd.lower() or keyword in desc.lower():
                results.append((category, cmd, desc))
    return results
```

### 快速发送

```python
def _on_send_selected_cmd(self):
    """发送选中的命令"""
    current = self._cmd_list.currentItem()
    if not current:
        self.log_warning("请先选中一个命令")
        return
    
    cmd = current.data(Qt.ItemDataRole.UserRole)
    self._send_command(cmd)
```
